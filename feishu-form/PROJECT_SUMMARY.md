## 🎯 设计亮点

### 1. 极致简化的扁平化设计
- ❌ 不做：子表、复杂联动、多层嵌套
- ✅ 只做：核心数据收集、简单统计

### 2. 3分钟创建原则
- 零学习成本的界面
- 直观的拖拽式编辑
- 即时预览和反馈

### 3. 高性能架构
- Fastify 后端（比 Express 快 2-3 倍）
- SQLite 同步操作（无异步复杂度）
- Vite 极速热更新
- Tailwind JIT 按需生成

### 4. 完整的数据流
```
创建表单 → 配置字段 → 分享链接 → 用户填写 → 数据收集 → 统计分析
```

## 🛠️ 技术栈

### 后端
- **Fastify** - 高性能 Node.js 框架
- **SQLite3** + **Better-SQLite3** - 轻量级数据库
- **@fastify/cors** - CORS 支持

### 前端
- **React 18** - 组件化 UI 框架
- **React Router 6** - 路由管理
- **Tailwind CSS v3** - 原子化样式
- **Vite** - 现代化构建工具

## 🚀 如何使用

### 快速启动（3步）

```bash
# 1. 进入项目目录
cd feishu-form

# 2. 安装依赖
cd backend && npm install
cd ../frontend && npm install

# 3. 启动服务
cd .. && ./start.sh
```

```
前端：http://localhost:3000
后端：http://localhost:3001
```
