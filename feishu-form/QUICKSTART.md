# 🚀 快速入门指南

欢迎使用**飞书表单**！这是一个极简、快速的协作表单系统。

## ⚡ 5分钟快速开始

### 第 1 步：安装依赖

```bash
# 进入项目目录
cd feishu-form

# 安装后端依赖
cd backend
npm install

# 安装前端依赖
cd ../frontend
npm install
```

### 第 2 步：启动服务

**方式 A - 一键启动（推荐）**
```bash
cd feishu-form
./start.sh
```

**方式 B - 分别启动**

打开两个终端：

```bash
# 终端 1 - 启动后端
cd feishu-form/backend
npm start

# 终端 2 - 启动前端
cd feishu-form/frontend
npm run dev
```

### 第 3 步：开始使用

1. 打开浏览器访问：`http://localhost:3000`
2. 点击"创建表单"
3. 添加字段，设置属性
4. 保存并分享给他人填写
5. 查看统计数据

## 🎯 核心功能演示

### 创建第一个表单

1. **设置基本信息**
   - 表单标题：客户满意度调查
   - 表单描述：感谢您的宝贵意见

2. **添加字段**
   ```
   ✅ 单行文本 - "您的姓名" (必填)
   ✅ 邮箱 - "联系邮箱" (必填)
   ✅ 单选 - "整体满意度"
      选项：非常满意 / 满意 / 一般 / 不满意
   ✅ 多选 - "改进方向"
      选项：界面设计 / 功能完善 / 性能优化
   ✅ 多行文本 - "详细建议"
   ```

3. **保存表单**
   - 点击"创建表单"按钮
   - 系统自动跳转到填写页面

### 填写表单

1. 填写所有必填字段
2. 点击"提交表单"
3. 看到提交成功页面

### 查看统计

1. 返回首页
2. 点击表单卡片的"统计"按钮
3. 查看：
   - 总回复数
   - 各字段统计分布
   - 具体回复列表

## ⚙️ 常用配置

### 修改端口

**后端端口** (backend/server.js):
```javascript
await fastify.listen({ port: 3001, host: '0.0.0.0' });
```

**前端端口** (frontend/vite.config.js):
```javascript
server: {
  port: 3000
}
```

### 数据库位置

数据库文件自动创建在：
```
backend/feishu-form.db
```

可以直接使用 SQLite 工具查看：
```bash
sqlite3 backend/feishu-form.db
```

## 🐛 常见问题

### Q: 端口被占用怎么办？
A: 修改配置文件中的端口号，或终止占用进程。

### Q: 依赖安装失败？
A: 切换 npm 源：
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 如何备份数据？
A: 复制数据库文件：
```bash
cp backend/feishu-form.db backup/feishu-form-backup.db
```

### Q: 可以部署到服务器吗？
A: 可以！使用 pm2 或 nginx + systemd 部署。
